<h2> 📘 Hướng dẫn sử dụng</h2>

- Tất cả các file hướng dẫn gốc dùng tiếng Việt với đuôi <code>.txt</code>.
- Mỗi file đều có thêm một bản tiếng Anh tương ứng với đuôi <code>.en.txt</code> nằm cùng thư mục.
- Bạn có thể mở các file này bằng bất kỳ editor nào (VS Code, nano, v.v.) để đọc và luyện tập các lệnh Git.
- Thư mục <b>Note_git</b> là phần lý thuyết, các thư mục <b>tutorial*</b> là phần thực hành (commit, branch, merge, rebase,...).
