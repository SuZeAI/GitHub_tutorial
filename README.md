<h1>🌱 Git For Everyone</h1>
<h2> 🔭 Description Github_tutorial</h2>

1. Folder Note_git: contains folders that guide you to learn Git

      - Folder local_repo: explains operations and commands in the local repository
      
      - Folder remote_repo: shows you how to connect with the remote (cloud) repository
      
      - Folder Grow: develops and extends Git commands
     
2. The other folders are for practicing Git
3. File test_git.txt

      - Summarizes the main Git commands
  
      - Notes and tips when coding

<h2> 📘 Usage Guide</h2>

- All original guide files are in Vietnamese with the <code>.txt</code> extension.
- Each file has a corresponding English version with the <code>.en.txt</code> extension in the same folder.
- You can open these files with any editor (VS Code, nano, etc.) to read and practice Git commands.
- The <b>Note_git</b> folder is for theory, while the <b>tutorial*</b> folders are for hands-on practice (commit, branch, merge, rebase, ...).
